# odin-library
Library created with HTML, CSS, JS

Features
Book Tracking: Users can add, view, and delete books from their personal library.
Responsive UI: Built with Bootstrap to ensure a clean and responsive interface across devices.
Add Books via Modal: Use a modal to add new books, including title, author, and page count.

Planned Features
Local Storage: Implementing local storage to persist books across sessions.
Google Books API Integration: Fetching book cover art using the Google Books API to display cover images for each book.
